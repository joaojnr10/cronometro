# ⏱️ Cronômetro Digital








Um cronômetro digital desenvolvido com HTML, CSS e JavaScript, capaz de marcar horas, minutos e segundos em tempo real.

O projeto foi criado para praticar lógica de programação, manipulação do DOM e controle de tempo com JavaScript, além de aplicar responsividade para diferentes dispositivos.

# 🚀 Acesse o projeto

🔗 Demo Online:
https://joaojnr10.github.io/cronometro/

# 📱 Funcionalidades

✔️ Iniciar o cronômetro
✔️ Pausar a contagem do tempo
✔️ Resetar o cronômetro
✔️ Contagem em horas, minutos e segundos
✔️ Interface responsiva para desktop, tablet e celular

# 🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript
